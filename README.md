# site_server
Сайт для публикации статей. 

Основной функционал:                             
-Поиск статей                          
-Добавление, редактирование и удаление статей           
-Комментарии к постам                  


Используемые технологии:                   
-Node.js                  
-JavaScript              
-Express                                   
-Mongo.db
